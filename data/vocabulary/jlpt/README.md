# JLPT N1–N5 词汇数据

本目录保存项目使用的 JLPT 分级词汇 CSV。

## 数据文件

- `jlpt-vocabulary.csv`：统一整理后的总表，按 N5、N4、N3、N2、N1 排列。
- `n1.csv` 至 `n5.csv`：上游仓库的各等级原始 CSV，保留用于追溯和后续更新。
- `SOURCE_LICENSE.txt`：上游仓库的 MIT 许可证原文。

总表字段：

| 字段 | 说明 |
| --- | --- |
| `id` | 项目内稳定标识，格式为 `jlpt-n{level}-{sequence}` |
| `level` | JLPT 参考等级 |
| `expression` | 日语常用表记 |
| `reading` | 假名读音 |
| `meaning_en` | 上游英文释义 |
| `meaning_zh` | DeepSeek V4 Pro 生成的简体中文释义 |
| `tags` | 上游标签原文 |
| `source_guid` | 上游 Anki 条目标识 |
| `source` | 上游 GitHub 仓库 |

## 重要限制

JLPT 官方在 2010 年改版后不再发布固定的考试词汇清单。因此，本目录中的等级是社区整理的学习参考，不应标注为“JLPT 官方词表”，也不能保证某个单词只会出现在对应等级的考试中。

`meaning_zh` 已全部补齐，但仍需逐条审核。全部 8,131 条中文释义于 2026-06-25 使用 `deepseek-v4-pro` 重新生成，输入包括日语表记、假名读音、JLPT 参考等级和英文参考释义。

生成工具为 `scripts/translate_vocabulary_deepseek.py`。提示词要求：

- 以表记和读音共同确定词条，严格处理同形异读词。
- 英文仅作为可能有误的参考，不盲目照译。
- 只保留适合 JLPT 学习的核心常用义。
- 识别前后缀、助数词、敬语以及及物、不及物等信息。
- 不混入例句、词源、人名、异体字说明或其他词典元数据。

生成前使用固定随机样本进行了两轮各 100 条的质量检查。最终版本样本中没有发现明显错误，8 条存在措辞或义项取舍方面的小问题。中文列应视为高质量待审核初稿，而不是出版级词典释义。

总表对原始数据做了两项保守整理：

- 为两个原本缺少 `reading`、且表记全部为假名的 N4 条目补上了读音。
- 保留跨等级重复词和同形同音的不同释义，不在缺少词典条目 ID 的情况下强行去重。
