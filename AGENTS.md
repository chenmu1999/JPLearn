# JPLearn 项目执行约束

本项目的环境边界、部署规范与虚拟机操作说明已统一整理到开发手册，**以该手册为准**：

➡ [`Docs/开发手册.md`](Docs/开发手册.md)

## 最高优先级（务必遵守）

Windows 本机工作区 `D:\Project\Web\JPLearn` **只用于书写、查看和修改源码与项目文档**。

禁止在本机安装依赖、编译、构建、检查、测试、数据库操作、调用 API、启动进程或部署。所有此类操作必须在 Ubuntu 虚拟机中完成。

修改源码后只能报告“源码已修改，尚未运行验证”。除非用户明确撤销，否则不得例外。

**如果你是 agent，或任何 AI 大模型，一律禁止自行开启 ngrok。** ngrok 隧道（对外公网访问）必须经用户明确授权才能启动，正常开发不需要开启。

完整说明见上述开发手册第一节《环境边界》。
