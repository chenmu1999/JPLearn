# 题型与练习设计

## 设计原则

JPLearn 不试图用一种练习覆盖所有知识点。每个知识点根据自身类型和能力目标选择合适题型，优先贴近 JLPT N5 真题格式，同时允许加入更适合长期学习的辅助题型。

核心原则：

- 真题格式优先：题型、说明、选项数量和考点尽量参考 N5。
- 知识点驱动：每道题都要明确关联一个或多个知识点。
- 逐知识点评分：每个受影响知识点独立判定正确或错误，正确 `+20`，错误 `-10`。
- 题型分阶段：先做文本题，再做听力；先做易生成题，再做复杂篇章题。
- 不强求所有知识点都有同一种练习。

## N5 真题题型

### 文字・語彙

- 汉字读音：给句子中的汉字词，选择平假名读法。
- 表记写法：给平假名词，选择正确写法。
- 语境选词：句子挖空，选择合适词汇。
- 近义改写：选择与原句意思相近的表达。

### 文法

- 语法填空：句子挖空，选择助词、活用或句型。
- 句子排序：排列词块，选择星号位置。
- 篇章语法：短文挖空，结合上下文选择语法项。

### 読解

- 短文理解：读短文、留言、对话，回答问题。
- 中篇理解：读较长文章，回答内容理解问题。
- 信息检索：读通知、时间表、广告、菜单，查找信息。

### 聴解

- 课题理解：听问题和对话，判断该做什么。
- 要点理解：听问题和对话，抓时间、地点、理由等重点。
- 发话表达：看图听问题，选最自然发话。
- 即时应答：听一句话，选自然回应。

## 知识点到题型映射

### 词汇知识点

优先题型：

- 汉字读音：适合有汉字表记的词。
- 表记写法：适合常见汉字词和假名词。
- 语境选词：适合名词、动词、形容词、副词。
- 近义改写：适合有清晰中文释义或常见替代表达的词。
- 造句使用：适合高频动词、形容词、常用名词。

辅助题型：

- 中文释义选择。
- 日文例句理解。
- 词性判断。

### 语法知识点

优先题型：

- 语法填空：适合助词、活用、句型、接续。
- 句子排序：适合固定结构、比较句、程度表达、时间表达。
- 篇章语法：适合连接、指代、顺序、原因、转折。
- 造句使用：适合需要主动掌握的句型。

辅助题型：

- 接续判断。
- 错误识别。
- 日中互译中的结构判断。

### 阅读类能力

阅读类能力不一定对应单个词汇或语法，而是可以关联多个知识点。

优先题型：

- 短文理解。
- 信息检索。
- 例句理解。

评分策略：

- 目标知识点理解正确：对应知识点加分。
- 明显误解某个词汇或语法：对应知识点扣分。
- 不对文章中所有词汇逐一评分，只评估目标知识点和明显暴露问题的知识点。

### 听力类能力

听力放在后续阶段。

优先题型：

- 要点理解。
- 即时应答。
- 发话表达。

前置条件：

- 音频切片或可定位播放。
- 听力原文可解析。
- 题目和答案可与音频片段稳定对应。

## 第一版题型优先级

### Phase A：文本基础题

先实现这些题型：

- 词汇语境选词。
- 词汇近义改写。
- 语法填空。
- 句子排序。
- 例句理解。
- 造句使用。

原因：这些题型不依赖音频切片，AI 生成和批改成本低，适合快速建立闭环。

### Phase B：阅读题

再实现：

- 短文理解。
- 信息检索。
- 篇章语法。

原因：这些题型需要生成或维护短文材料，关联多个知识点，数据结构和 UI 稍复杂。

### Phase C：听力题

最后实现：

- 课题理解。
- 要点理解。
- 发话表达。
- 即时应答。

原因：听力需要音频切片、播放控件、听力原文对齐和更复杂的资源管理。

## 题目生成约束

- 每道题必须有 `targetKnowledgePointId`。
- 每道题可以关联多个辅助知识点。
- 题目难度以 N5 为基准，不主动引入 N4 以上语法。
- 多选题默认 4 个选项；听力即时应答可使用 3 个选项，贴近真题。
- 干扰项必须合理，但不能过分超纲。
- 题目解析要说明为什么正确，也要指出错误选项的关键问题。

## 评分规则

- 所有题型共用同一掌握规则。
- 对某知识点判定正确：`+20`。
- 对某知识点判定错误：`-10`。
- 分数范围固定为 `0` 到 `100`。
- 达到 `100` 标记为已掌握。
- 不设置部分正确。

## 实现建议

题型层面建议引入 `ExerciseType`，不要只依赖 `PracticeMode`。

`PracticeMode` 表示大类：

- `SENTENCE_WRITING`
- `COMPREHENSION`

`ExerciseType` 表示具体题型：

- `VOCAB_CONTEXT_CHOICE`
- `VOCAB_REPHRASE`
- `GRAMMAR_FILL_BLANK`
- `GRAMMAR_SENTENCE_ORDER`
- `EXAMPLE_COMPREHENSION`
- `SHORT_READING`
- `INFO_SEARCH`
- `SENTENCE_WRITING`

听力题型后续再扩展。
