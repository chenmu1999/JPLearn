# 五十音与假名输入学习设计

## 阶段目标

这一阶段不以背完五十音图为终点，而以实际输入能力为终点。

完成本阶段后，用户应当能够：

- 看到任意平假名或片假名并识别其读音。
- 根据读音回忆并输入对应的平假名或片假名。
- 使用罗马音输入法输入任意常见假名形式的日语词。
- 使用日语 IME 的直接假名输入方式输入任意常见假名形式的日语词。
- 正确处理浊音、半浊音、拗音、促音、长音和拨音。
- 在平假名与片假名之间切换，不依赖五十音表完成输入。

本阶段只评价假名识别与输入，不要求用户理解单词含义、汉字写法或语法。

## 能力拆分

### 识别

- 假名到读音：看到 `か` 能想到 `ka`。
- 读音到假名：看到或听到 `ka` 能想到 `か` 或 `カ`。
- 平片假名对应：知道 `か` 与 `カ` 表示同一个音。

### 罗马音输入

- 输入清音、浊音和半浊音。
- 输入拗音和其他小假名组合。
- 输入促音、拨音和长音。
- 了解输入法允许的常见罗马音写法，但练习使用一套稳定的推荐写法。

### 假名输入

- 熟悉日语假名键位。
- 不依赖屏幕键位提示完成单个假名输入。
- 连续输入假名单词，并正确使用浊点、半浊点和小字切换。
- 逐步减少看键盘和逐字寻找按键的行为。

## 学习顺序

### 模块 1：清音与基本键位

按行学习平假名清音：

1. あ行
2. か行
3. さ行
4. た行
5. な行
6. は行
7. ま行
8. や行
9. ら行
10. わ行与 `ん`

每引入一组新假名，立即进行四种练习：

- 假名选读音。
- 读音选假名。
- 罗马音输入假名。
- 假名键位输入假名。

每组练习混入已经学过的假名，避免用户只靠题目顺序记忆。

平假名清音达到基本熟练后，引入对应片假名。片假名不重新作为一套完全独立的表死记，而是通过平假名、片假名和读音之间的转换建立联系。

### 模块 2：浊音与半浊音

训练以下对照：

- `か / が`
- `さ / ざ`
- `た / だ`
- `は / ば / ぱ`

练习重点：

- 识别清音与浊音的差别。
- 使用罗马音输入 `ga`、`za`、`da`、`ba`、`pa` 等音。
- 使用假名输入中的浊点与半浊点。
- 在相近假名混合出现时保持准确。

### 模块 3：拗音与小假名

学习 `ゃ`、`ゅ`、`ょ`、`ぁ`、`ぃ`、`ぅ`、`ぇ`、`ぉ`、`ゎ` 以及片假名对应形式。

重点组合包括：

- `きゃ / きゅ / きょ`
- `しゃ / しゅ / しょ`
- `ちゃ / ちゅ / ちょ`
- `にゃ / にゅ / にょ`
- `ひゃ / ひゅ / ひょ`
- `みゃ / みゅ / みょ`
- `りゃ / りゅ / りょ`
- 对应的浊音和半浊音组合。
- 片假名外来语中常见的 `ファ`、`ティ`、`ディ`、`チェ` 等组合。

罗马音输入优先教授稳定、常见的输入方式，例如 `kya`、`shu`、`cho`；必要时再说明 `x` 或 `l` 前缀输入独立小假名的方法。

### 模块 4：特殊输入规则

集中训练容易造成实际输入错误的规则：

- 促音 `っ / ッ`：通过重复后续辅音输入，例如 `gakkou`。
- 拨音 `ん / ン`：根据后续字符练习 `n` 与 `nn` 的使用。
- 长音：平假名词中的元音组合，以及片假名中的长音符 `ー`。
- 独立小假名：使用 `x` 或 `l` 前缀输入。
- 容易混淆的罗马音输入，如 `じ`、`ぢ`、`ず`、`づ`。

这一模块以完整词语为主要练习单位，不再只输入单个假名。

### 模块 5：真实词语连续输入

从已经学过的假名中组合真实日语词，让用户建立连续输入能力。

词目应逐步覆盖：

- 只有清音的词，如 `さかな`。
- 含浊音或半浊音的词，如 `かばん`、`パンダ`。
- 含拗音的词，如 `きょう`、`しゅくだい`。
- 含促音的词，如 `きって`、`がっこう`。
- 含拨音的词，如 `しんぶん`。
- 含长音的片假名词，如 `コーヒー`、`スーパー`。
- 同时包含多种规则的词。

同一个目标词分别进入罗马音输入和假名输入练习。两种模式分别记录熟练度，不能用其中一种的成绩替代另一种。

### 模块 6：平假名与片假名混合

训练用户在两套假名之间快速切换：

- 根据读音输入指定的平假名或片假名。
- 将平假名词转换为片假名输入，反向亦然。
- 输入包含平假名和片假名的短语。
- 根据词语类型选择合适的假名形式。

本模块只要求形式选择正确，不扩展到汉字转换能力。

## 单次学习流程

一次练习建议控制在 5 至 10 分钟：

1. 复习上一轮最不熟练的假名或输入规则。
2. 引入少量新假名或一个新输入规则。
3. 完成识别练习。
4. 完成罗马音输入练习。
5. 完成假名键位输入练习。
6. 使用真实词语完成综合输入。
7. 展示本轮错误类型和下一轮复习目标。

新内容每轮不宜过多。学习重点从“答对”逐渐转为“无需提示、连续输入、保持准确”。

## 练习界面

练习页需要提供：

- 目标内容：假名、罗马音、读音提示或音频中的一种。
- 输入区域：直接接收用户通过系统 IME 输入的结果。
- 输入模式切换：罗马音输入、假名输入。
- 假名范围切换：平假名、片假名、混合。
- 即时反馈：正确字符、错误字符和漏输位置。
- 可选键位提示：仅在假名输入的早期阶段显示。
- 下一题、重试和播放读音操作。

系统判断最终输入的假名结果，同时记录输入过程。罗马音输入练习还应根据按键序列判断用户是否掌握目标输入规则，而不能只判断 IME 转换后的最终文字。

## 提示机制

提示应逐级出现：

1. 不显示提示。
2. 显示读音或罗马音首字母。
3. 显示完整罗马音。
4. 假名输入模式下高亮目标键位。
5. 显示完整答案并要求重新输入。

使用提示后可以完成题目，但不计为独立掌握。系统应记录用户使用了哪一级提示。

## 错误分类

系统至少区分以下错误：

- 假名识别错误。
- 平假名与片假名混淆。
- 清音、浊音或半浊音混淆。
- 大假名与小假名混淆。
- 促音输入错误。
- 拨音输入错误。
- 长音输入错误。
- 罗马音按键序列错误。
- 假名键位错误。
- 漏输、重复或顺序错误。

错误分类用于安排后续练习，不只展示“正确”或“错误”。

## 熟练度记录

熟练度分为三个层面：

- 假名识别熟练度。
- 罗马音输入熟练度。
- 假名输入熟练度。

每个假名和特殊输入规则分别记录：

- 作答次数。
- 独立正确次数。
- 使用提示次数。
- 最近错误类型。
- 最近练习时间。
- 平均输入耗时。

准确率优先于速度。只有在不使用提示且持续正确后，才逐步提高速度要求。

## 阶段验收标准

用户需要分别通过罗马音输入和假名输入验收。

验收题目应包含未在近期练习中重复出现的真实词语，并覆盖：

- 平假名和片假名。
- 清音、浊音和半浊音。
- 拗音和小假名。
- 促音。
- 拨音。
- 长音。
- 多种规则组合。

建议通过条件：

- 单个假名识别正确率不低于 95%。
- 罗马音输入综合词语正确率不低于 95%。
- 假名输入综合词语正确率不低于 90%。
- 全程不使用答案或键位提示。
- 不存在持续无法输入的假名或特殊规则。

通过后，用户应能在系统给出任意常见假名词时，独立选择两种输入方式中的任意一种完成输入。

## 暂不纳入本阶段

- 汉字输入与候选词选择。
- 单词释义记忆。
- 音调和口语发音评分。
- 手写假名识别。
- 日语句子和语法输入。
- 输入速度竞赛或排行榜。
